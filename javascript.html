目前 Puerh 的大部分组件都使用 Twitter Bootstrap 库的自定义 Javascript 插件。删除了其中网站不常用的 Carousel 等组件，修改 Tips、Modals 等插件以适合百姓网需求，增加 Validator 和 Linkage 组件。

Modals(弹出层) Dropdowns（下拉菜单） Tabs（标签切换） Tips（浮层提示条）
Alert（可关闭消息） Collapse（折叠） Typeahead（自动完成） Scrollspy（滚动生成）
Validator（表单验证） Linkage（联动操作） Lightbox（图片浏览浮层）
注：API 接口使用可以参考：http://twitter.github.com/bootstrap/javascript.html
